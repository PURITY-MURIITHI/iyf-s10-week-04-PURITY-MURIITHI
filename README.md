## Project Description
This week focused on javascript basics from understanding variables, datatypes, operators to writing functions and control flows.
We also got to work with arrays, array methods, objects and manipulate data. This project uses the skills learnt to build an interactive calculator and a student grade tracker.

## Technologies Used
- HTML5
- JavaScript
- Git and Github
- VS Code

## Features
- Perform arithmetic operations using a calculator
- Handle edge cases like division by zero
- Store and manage student data using objects
- Calculate student and subject averages
- Identify top-performing and struggling students
- Generate formatted report cards
- Use array methods (`map`, `filter`, `reduce`, `find`)
